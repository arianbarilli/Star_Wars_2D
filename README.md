# Star Wars 2D  🐍

Desenvolvido Por Arian Barilli /RA 1129868 & João Pedro Voginski Gonçalves /RA 1129618 

Jogo feito por fans de Star Wars, inspirado nas batalhas de caças estelares.

Neste Jogo o Objetivo é controlar seu caça para sobreviver em meio a uma batalha
no espaço em um Caça Rebelde X-Wing Modelo T-65.
